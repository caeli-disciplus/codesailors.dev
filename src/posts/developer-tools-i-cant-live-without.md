A curated list of essential tools, extensions, and utilities that boost productivity and make development work more enjoyable.

## Editor: VS Code

The ecosystem of extensions is unmatched. Paired with GitHub Copilot, it dramatically reduces the mechanical parts of coding.

## Terminal: Warp

A modern terminal that actually remembers your command history intelligently and lets you edit commands like a text editor.

## HTTP Client: Bruno

A local-first, git-friendly alternative to Postman. API collections live as plain files in your repo — no cloud account required.

## Browser DevTools

Still underutilized by most developers. The Performance and Memory panels alone can diagnose 80% of frontend issues without any external tooling.
