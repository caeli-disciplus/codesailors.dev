Mastering Flexbox, Grid, and other layout methods to create responsive, maintainable user interfaces.

## Flexbox for One-Dimensional Layouts

Flexbox excels at distributing space along a single axis — use it for navigation bars, card rows, and centering elements. `justify-content` and `align-items` replace a decade of float hacks.

## CSS Grid for Two-Dimensional Layouts

Grid handles rows *and* columns simultaneously. Define your layout in one place with `grid-template-areas` and let content flow naturally into named regions.

## Container Queries

The next evolution beyond media queries — style components based on their own container's size, not the viewport. Now widely supported and production-ready.

## Subgrid

Align nested elements to a parent grid's tracks. Eliminates the need for duplicated gap values and makes complex card layouts trivial.
